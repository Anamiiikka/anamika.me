# Anamika's Portfolio

A personal portfolio website for a Product Designer & Dog Whisperer, built with Next.js and Tailwind CSS.

## Features

- **Custom Design**: Matches the specific dark theme and typography requested.
- **Responsive**: Adapts to different screen sizes.
- **Next.js 15+**: Uses the latest App Router features.
- **Tailwind CSS**: Utility-first styling.
- **Custom Fonts**: Uses `Permanent Marker` and `Poppins` from `next/font/google`.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app/page.tsx`: The main landing page with the portfolio content.
- `src/app/layout.tsx`: Root layout containing font definitions and global styles.
- `src/app/globals.css`: Global CSS variables and Tailwind directives.

## Customization

To update the content, edit `src/app/page.tsx`.
To change fonts or global styles, edit `src/app/layout.tsx` or `src/app/globals.css`.
